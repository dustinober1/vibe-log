# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-12

### Added
- Initial release of log-vibe
- Beautiful, contextual logging with color-coded levels
- Zero runtime dependencies
- Full TypeScript support with comprehensive type definitions
- Dual module support (CommonJS and ES Modules)
- Five log levels: debug, info, success, warn, error
- Scoped loggers with `createScope()`
- Pretty printing for objects, arrays, and errors
- Circular reference detection in object printing
- Configurable logger with `configure()` function
- Automatic color support detection for terminals
- ASCII icon fallback for non-Unicode terminals
- Input validation for context and message parameters
- Log level filtering
- Timestamp support with multiple formats (time and ISO 8601)
- Comprehensive JSDoc documentation
- 30+ unit tests with Vitest
